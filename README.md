# 🏝️ 폭풍에 떠밀려 온 섬 (Storm-Drifted Island)

> **"폭풍에 휩쓸려 온 외딴섬에서 빚을 상환하고, 진정한 섬의 주인이 되어라!"**
> 어느 날 갑자기 거대한 폭풍우에 휘말려 이름 모를 외딴섬에 표류하게 된 주인공. 그곳에서 만난 괴짜 노인의 도움으로 작은 텐트 한 켠을 얻게 되지만, 섬에서 나가기 위해서는 노인에게 총 100,000골드의 빚을 갚아야 합니다. 농사를 짓고, 바다낚시를 즐기며, 가축을 기르고, 주방에서 요리를 만들어 판매하며 빚을 청산하고 섬의 진짜 주인이 되는 농장 시뮬레이션 게임입니다.

<br>

## 🚀 1. 실행 방법 (How to Run)

본 게임은 **Docker 환경**을 통해 어디서나 환경의 구애 없이 동일하게 실행할 수 있도록 패키징되어 있습니다. 아래 단계를 순서대로 진행해 주세요.

### (1) Docker 이미지 풀 (Pull Docker Image)
```bash
docker pull --platform linux/amd64 peg0525/team21_project:0.1.0
```

### (2) 애플리케이션 실행 (Run the App)

* **옵션 A — 자동 실행 (Run Automatically):** 스크립트를 사용하여 컨테이너를 생성함과 동시에 게임을 자동 실행합니다.
  ```bash
  scripts/run.sh
  ```

* **옵션 B — 컨테이너 진입 후 수동 실행 (Manual Launch):** 컨테이너 내부의 Bash 셸 환경으로 먼저 진입한 뒤, 개발 환경 내에서 직접 게임을 제어하고 수동으로 실행합니다.
  ```bash
  scripts/run_shell.sh
  ```
  셸 환경에 진입한 후 컨테이너 내부 터미널에서 다음 명령어를 입력.
  ```bash
  ./build/main
  ```

### (3) 웹 브라우저 접속 및 플레이 (Web Browser VNC)
게임이 성공적으로 구동되면 크롬(Chrome) 등의 웹 브라우저를 실행하고 아래 주소로 접속하여 GUI 환경에서 플레이를 시작합니다.
```text
http://localhost:6080/vnc.html
```

## ⚙️ 3. 핵심 게임 시스템 (Core Game Systems)

### ⏱️ (1) 실시간 시간 및 낮밤 시스템
게임 내 세계는 현실 시간과 유기적으로 연동되어 흐르며, 시간에 따라 행동 제약이 발생합니다.
* **시간 스케일:** 현실 시간 **10분**이 게임 내의 **하루(1일)**로 변환됩니다. (낮 7분, 밤 3분)
* **환경 변화:** 낮과 밤이 전환될 때 필드의 배경 그래픽과 조명이 실시간으로 변경됩니다.
* **밤의 행동 제약:** 밤이 되면 **항구(상점 및 판매 기능)가 전면 폐쇄**됩니다. 밤에는 오직 농사(물주기/수확), 목장 관리, 주방 요리만 가능하며, 집 안의 침대에서 **[취침]**을 선택하여 다음 날 아침으로 즉시 시간을 건너뛸 수 있습니다.

### 📻 (2) 아침 방송 및 날씨·이벤트 시스템
매일 새로운 낮이 시작될 때마다 게임 진행이 잠시 멈추고 **'괴짜 노인의 아침 방송'** 화면이 자동으로 등장(스킵 불가)하여 날씨와 무작위 이벤트를 공지합니다.

* **날씨 분포 및 효과**:
  * ☀️ **맑음:** 기본 상태이며, 농사와 야외 활동에 적합합니다.
  * ☁️ **흐림:** 하늘이 흐려지며, 낚시터에서 희귀/전설 물고기가 등장할 확률이 소폭 상승합니다.
  * 🌧️ **비:** 대지에 비가 내려 **밭에 심어진 모든 작물이 즉시 성장을 완료**하고 수확 가능 상태가 됩니다.
  * 🏜️ **가뭄:** 극심한 가뭄으로 인해 밭에 물을 주더라도 수확량 2배 증가 버프 효과가 완벽히 무력화됩니다.

* **랜덤 이벤트 (하루 최대 1개 발생)**:
  * 🐗 **멧돼지 습격:** 야생 멧돼지가 밭을 헤집어 현재 자라고 있는 **작물의 절반(소수점 버림)이 파괴**됩니다.
  * 🦠 **전염병 발생:** 목장에 돌발 전염병이 감염되어 보유 중인 **가축의 절반(소수점 버림)이 폐사**합니다.
  * 🏴‍☠️ **해적 침공:** 섬에 해적이 들이닥쳐 창고에 보관 중이던 **섬 특산물 스토리지 물품이 전부 소멸**합니다.
  * 🌾 **풍년 선언:** 대지의 기운이 넘쳐 그날 수확하는 모든 작물의 수확량이 **3배**로 증가합니다.
  * 🐟 **물고기 떼 출현:** 해안가에 물고기 떼가 몰려와 낚시 성공 시 획득 수량이 **2배**가 됩니다.
  * 💎 **기타 혜택:** 행상인 방문(희귀 레시피 판매 및 특산물 2배가 매입), 보물 발견(최대 2,000G 무작위 획득), 금덩이 발견(5,000G 즉시 지급)이 있습니다.

## 📊 4. 세부 게임 플레이 데이터 (Gameplay Details)

### 🌾 농사 데이터 (Farming)
메인 홈 화면(30x24 타일)에 위치한 4x4 크기의 밭(농장 스킨 장착 시 5x5로 확장)에서 작물을 재배합니다. 빈 칸에서 인벤토리를 열어 모종을 심은 뒤 매일 물을 주면 수확량이 2배가 됩니다. 1회성 아이템인 **'스프링클러(500G)'**를 소비하면 밭 전체에 즉시 물을 줄 수 있습니다.

| 작물 종류 | 모종 구매가 | 성장 소요 기간 | 기본 수확량 | 물 준 날 수확량 | 최종 판매가 |
| :---: | :---: | :---: | :---: | :---: | :---: |
| **밀** | 30 G | 1일 | 1개 | 2개 | **50 G** |
| **상추** | 30 G | 2일 | 1개 | 2개 | **70 G** |
| **토마토** | 30 G | 3일 | 1개 | 2개 | **100 G** |
| **감자** | 30 G | 4일 | 1개 | 2개 | **150 G** |

### 🎣 낚시 미니게임 (Fishing)
낚시터에서 Space 키를 누르면 타이밍 게이지 바가 작동합니다. 좌우로 왕복하는 바를 타깃 지점에 맞춰 다시 Space 키로 멈추어 초록 타깃 내에 멈추게 되면 낚시에 성공하게 됩니다.
* **어종 정보**:
  * **고등어 (일반):** 항상 출현. 기본 확률 80%. 판매가 **100 G**
  * **연어 (희귀):** 비 오는 날 출현 확률 상승(15% → 25%). 판매가 **300 G**
  * **황금참치 (전설):** 비 또는 흐린 날씨 조건 충족 시 제한적 출현(최대 15%). 판매가 **800 G**

### 🪵 목장 시스템 (Ranching)
가축은 별도의 먹이 공급 없이 매일 산물을 자동으로 생산합니다. 최대 수용량은 16마리입니다.

* **닭 (구매가: 500G):** 매일 **달걀 1개**(판매가 60G)를 확정 생산합니다.
* **돼지 (구매가: 800G):** 추후 업데이트를 통해 도축 및 정육 판매 시스템을 구상 중입니다.
* **소 (구매가: 1,200G):** 매일 **우유 1개**(판매가 80G)를 자동 생산합니다.

### 🎰 도박장 슬롯머신
딜러 '조커'를 통해 하루 최대 5회까지만 배팅 가능한 미니게임입니다. `100G / 500G / 1,000G` 중 골드를 걸면 슬롯 3개가 회전하기 시작하며, Space 키로 좌->중->우 순서대로 하나씩 정지시킵니다.
* **3개 심볼 전원 일치:** 대박 판정, **베팅 금액의 5.0배** 지급
* **2개 심볼 일치:** 아깝네요 판정, **베팅 금액의 1.5배** 지급
* **심볼 불일치:** 뱅크럽, 베팅 금액 전액 소멸
* *슬롯 심볼 종류(7종):* 감자, 상추, 토마토, 우유, 밀, 연어, 달걀

### 📉 특산물 매매 경제 (주식 시스템)
매일 아침 노인이 당일 책정된 특산물의 고유 단가를 제시하며 [구매/패스] 여부를 묻습니다. 구매 창 옆에는 **최근 7일간의 변동 가격 추이가 담긴 꺾은선 그래프 UI**가 표출됩니다. 특산물은 인벤토리 한도 내에서 영구히 썩지 않으므로 전략적 저점 매수 고점 매도가 필수적입니다.
* **황금조개:** 변동 주기가 짧고 리스크가 적음. 초기 가격 500G (전일 대비 0.8배 ~ 1.2배 변동)
* **야생버섯:** 중간 수준의 가격대와 정형적 추세. 초기 가격 1,000G (전일 대비 0.6배 ~ 1.4배 변동)
* **신비초:** 하이 리스크 하이 리턴, 예측 불허형. 초기 가격 2,000G (전일 대비 0.2배 ~ 1.8배 변동)
